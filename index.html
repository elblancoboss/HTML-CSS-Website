WEEK 9. HTML and CSS forms

Published on January 5, 2015

Making CSS form was a great thing to learn and to add to my learning journal, I will be using this form page to allow my learning journal visitors to send me any feedback regarding my learning journal. Inserting a form on the webpage was not very tough as I had to use the form tag in order to insert a form. Adding form elements to the contact form was an essential part to build up my contact form. Adding elements such as the name, email address and website address for the user to fill in his/her details so it will enable me to contact them regarding their feedback or at least to make me aware of who there are. Using a fieldset really separates out the different sections of a form and it makes the form more orientated and organised. I also learnt how to add a text area and how to set up its size, the comment area enables the user to post any comments he/she has regarding my learning journal. Radio group was a nice element to add to my contact form which I used to ask the user if they prefer to receive more information from me, and at the end I inserted a submit input element which is a submit button to submit the form to me. For my stylesheet I used the form styling radio and submit input types in order to make them inputs. CSS forms are essential and they can be used however the user wants and they do not really have to be contact forms.
